# 만복사 살인사건

김시습의 『만복사저포기』를 현대적으로 재해석한 미스터리 소설입니다.

## 작품 소개

조선시대 고전 소설 『만복사저포기』의 배경인 남원 만복사를 무대로 한 현대 미스터리 소설입니다. 원작의 환상적이고 신비로운 분위기를 현대적 감각으로 재해석하여, 과거와 현재가 교차하는 신비로운 이야기를 그려냅니다.

## 주요 특징

- 고전 문학의 현대적 재해석
- 미스터리와 호러 장르의 결합
- 한국적 정서와 전통 문화의 표현
- 과거와 현재의 교차 구조
